### Controls
- Z, Arrow Up: Move up
- Q, Arrow Left: Move left
- S, Arrow Down: Move down
- D, Arrow Right: Move right
- Space: Bicycle
- E: Interact
- Escape: Menu

## Technologies Used

- React
- Vite
- Phaser
- Bun

## Run it locally

To launch the game:

```bash
git clone
cd pokemon
bun install
bun dev
```
